# Security policy

WW3MOD is a game mod, so most "security" issues will be crashes, save-file corruption, or multiplayer desyncs rather than classical vulnerabilities. Even so, please report concerning issues privately if they could be abused — for example, a crafted network packet that crashes an opponent's game.
